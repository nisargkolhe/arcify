Set Up Keyboard Shortcuts

Configure keyboard shortcuts to quickly access Arcify features from anywhere in your browser.

How to set up keyboard shortcuts:

  1. Open Chrome Extensions page (chrome://extensions/)
  2. Find Arcify in the list of extensions
  3. Click on "Details" for Arcify
  4. Scroll down to "Keyboard shortcuts"
  5. Click "Configure" next to "Toggle side panel"
  6. Set your preferred shortcut (e.g., Cmd/Ctrl + A)
  7. Click "Save" to apply the changes

Recommended Shortcuts:

  • üéØ Toggle Sidepanel: Cmd/Ctrl + S
  • üîç Spotlight: Cmd/Ctrl + T
